Blend Modes

Blending modes refers to how overlapping visuals will be blended together. Blend modes are used to manipulate and elevate photography and when experimented with can create interesting results.


Software like Adobe Photoshop views a pixel’s colors as a number value. The blend mode names relate to how the color’s number will be blended. For example the mode “multiply” will take the color values from the overlapping layer and the base layer and then multiply the color values to generate the output.


Experiment with blending modes in the interaction to see how colors will be blended together. Learn more about what you can do with blending modes and how each blend mode works using the resources below.
